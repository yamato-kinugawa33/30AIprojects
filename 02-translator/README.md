# 闇の翻訳機 (Dark Translator)

## 概要
このツールは、入力された平凡な言葉を、選ばれし者のみが理解できる「中二病」の言葉へと変換する禁断の魔導具（翻訳機）である。
OpenAIの叡智（API）を借り受け、システムプロンプトという名の契約によって、その人格を「中二病の男性」へと変貌させている。

## 特徴
- **中二病翻訳**: あらゆる言葉を深淵なる表現へと昇華させる。
- **安定した出力**: Temperatureを0.0に設定することで、混沌の中にも秩序ある回答をもたらす。
- **エラーハンドリング**: 万が一、運命の歯車が狂った（エラーが発生した）場合でも、世界観を崩さずに状況を伝える。

## 必要要件
- Python 3.x
- OpenAI API Key

## インストール方法

1. このリポジトリを召喚（クローン）する。
2. 必要な魔導書（ライブラリ）をインストールする。
   ```bash
   pip install -r requirements.txt
   ```
3. `.env` ファイルを作成し、あなたの真名（APIキー）を刻む。
   ```
   OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxx
   ```

## 使用方法

以下の呪文（コマンド）を唱えて起動せよ。

```bash
python main.py
```

- `You:` と表示されたら、翻訳したい言葉を入力せよ。
- `exit` または `quit` と入力することで、現世へと帰還（終了）できる。
