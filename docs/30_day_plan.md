# 30日間 AIアプリ開発カリキュラム

このカリキュラムは、AI開発の初心者から中級者レベルへとステップアップし、30個の多様なアプリケーションポートフォリオを作成することを目的としています。

## フェーズ 1: 基礎 & テキスト処理 (Days 1-5)
焦点: OpenAI API / Gemini API の基礎、プロンプトエンジニアリング、シンプルなUI (Streamlit/Console)。

1.  **01-simple-chat**: 基本的なチャットボット (GPT-4o/Gemini)
    -   *目的*: API認証、リクエスト/レスポンス構造の理解。
2.  **02-translator**: 多言語翻訳機
    -   *目的*: システムプロンプト、Temperature設定の理解。
3.  **03-summarizer**: 長文要約ツール
    -   *目的*: トークン制限の扱い、指示チューニング。
4.  **04-sentiment**: 感情分析ツール (ポジティブ/ネガティブ/中立)
    -   *目的*: 出力の解析 (JSONモード)、分類タスク。
5.  **05-keyword-extractor**: SEOキーワード抽出ツール
    -   *目的*: 構造化データの抽出。

## フェーズ 2: メディア & マルチモーダル (Days 6-10)
焦点: 画像生成、音声処理、画像認識 (Vision)。

6.  **06-image-gen**: テキスト to 画像生成 (DALL-E 3)
    -   *目的*: 画像生成API、プロンプトの拡張。
7.  **07-vision-analyzer**: 画像コンテンツ分析 (GPT-4o Vision)
    -   *目的*: マルチモーダル入力 (テキスト + 画像)。
8.  **08-tts-app**: テキスト読み上げプレーヤー
    -   *目的*: 音声API、バイナリレスポンスの扱い。
9.  **09-transcriber**: 音声文字起こし (Whisper)
    -   *目的*: 音声ファイルアップロード、Speech-to-Text。
10. **10-podcast-gen**: ミニポッドキャスト生成 (台本作成 + 音声合成)
    -   *目的*: 複数のAPIコールの連鎖 (LLM -> TTS)。

## フェーズ 3: RAG & データ活用 (Days 11-15)
焦点: Embeddings (埋め込み表現)、ベクトルデータベース、データとの対話。

11. **11-pdf-chat**: PDFとチャット (シンプルなRAG)
    -   *目的*: テキスト抽出、チャンク分割、コンテキスト注入。
12. **12-web-qa**: URL指定Web Q&A
    -   *目的*: Webスクレイピング + LLMによる要約・回答。
13. **13-embedding-search**: 意味検索エンジン (Semantic Search)
    -   *目的*: Embeddingsの理解 (キーワード検索との違い)。
14. **14-csv-analyzer**: CSVデータ分析アシスタント
    -   *目的*: コード生成 (LLMにPythonコードを書かせて分析)。
15. **15-youtube-summary**: YouTube動画要約
    -   *目的*: 字幕API + 要約。

## フェーズ 4: エージェント & ツール利用 (Days 16-20)
焦点: Function Calling、ツール利用、自律エージェント。

16. **16-weather-agent**: お天気ボット (Function Calling)
    -   *目的*: ツールの定義、引数の解析。
17. **17-search-agent**: Web検索エージェント (Tavily/SerpAPI)
    -   *目的*: 外部のリアルタイムデータへの接続。
18. **18-email-drafter**: トーン調整機能付きメール下書き生成
    -   *目的*: ユーザー設定の反映、テンプレート充填。
19. **19-task-planner**: AIタスク分解 & スケジューラー
    -   *目的*: 複雑な推論、構造化された出力。
20. **20-personal-tutor**: クイズ生成 & 採点チューター
    -   *目的*: 対話状態の管理 (スコアの保持など)。

## フェーズ 5: 発展的なアプリケーション (Days 21-25)
焦点: 複雑なワークフロー、メモリ、特化型タスク。

21. **21-resume-optimizer**: 履歴書 vs 求人票マッチング
    -   *目的*: 比較ロジック、具体的なアドバイス生成。
22. **22-code-reviewer**: Pythonコードレビュアー & リファクタリング
    -   *目的*: コード理解、Diff生成。
23. **23-blog-writer**: 本格ブログ記事生成 (構成案 -> 執筆 -> 推敲)
    -   *目的*: 長文生成戦略 (Chain of Thought)。
24. **24-sql-generator**: 自然言語 to SQL 変換
    -   *目的*: ドメイン特化言語への翻訳。
25. **25-voice-assistant**: 音声対話アシスタント (STT -> LLM -> TTS)
    -   *目的*: リアルタイム(に近い)会話ループ。

## フェーズ 6: ポートフォリオ & プロダクション (Days 26-30)
焦点: 仕上げ、デプロイ、UI/UX、最終ショーケース。

26. **26-dream-interpreter**: 夢占い & ビジュアル生成
    -   *目的*: クリエイティブライティング + 画像生成の組み合わせ。
27. **27-travel-planner**: ビジュアル旅行プランナー
    -   *目的*: リッチなUI、地図連携 (概念実証)。
28. **28-story-maker**: インタラクティブ・ストーリーメーカー
    -   *目的*: 動的な物語展開、ユーザー選択。
29. **29-ai-avatar**: シンプルAIアバター (動画/アニメーション概念)
    -   *目的*: 高度なメディア統合。
30. **30-portfolio-site**: ポートフォリオサイト本体
    -   *目的*: 全29アプリのショーケース、メタプロジェクト。
