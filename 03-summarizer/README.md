# 03-Summarizer (長文要約ツール)

OpenAI APIを利用した長文要約ツールです。トークン制限を超える長いテキストでも、自動的に分割して要約を行い、重要なポイントを抽出します。

## 主な機能

- **長文対応**: コンテキストウィンドウ（トークン制限）を超える長いテキストも、自動的に分割して処理します。
- **トークン最適化**: `tiktoken` ライブラリを使用し、モデルに合わせて正確にトークン数を計算して分割します。
- **指示チューニング**: システムプロンプトにより、編集者として振る舞い、簡潔な箇条書き形式で出力するように調整されています。

## 必要条件

- Python 3.x
- OpenAI API Key

## インストール

必要なライブラリをインストールします。

```bash
pip install openai python-dotenv tiktoken
```

`.env` ファイルを作成し、OpenAI APIキーを設定してください。

```text
OPENAI_API_KEY=your_api_key_here
```

## 使用方法

以下のコマンドでツールを起動します。

```bash
python main.py
```

起動後、要約したいテキストを入力してください。
終了する場合は `exit` または `quit` と入力します。

## 仕組み

1. **入力**: ユーザーからテキストを受け取ります。
2. **分割 (Chunking)**: テキストを `gpt-4o-mini` モデルのトークン数に基づいて安全なサイズに分割します。
3. **要約 (Map)**: 各チャンクに対して、「重要なポイントを箇条書きにする」という指示の元、要約を実行します。
4. **出力**: 分割された要約結果を結合して表示します。
